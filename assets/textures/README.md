# Texture Asset Layout

This directory stores runtime texture assets generated for the renderer.

Recommended structure:

```text
assets/textures/
├── inflation/
├── qgp/
├── plasma/
├── dark_ages/
├── reionization/
└── structure/
```

Recommended suffixes:

- `_albedo`
- `_emission`
- `_mask`
- `_density`
- `_lookup`

Preferred formats:

- `png` for masks and LDR maps
- `exr` for HDR or scalar-field data
- `pgm` for simple grayscale lookup maps generated by small offline tools

Source `.blend` files stay under `src/shaders/blender/`.

Current example:

- `structure/volume_macro_lookup.pgm` is the first Blender-driven lookup integrated into `src/shaders/volume.frag`.
- `dark_ages/gas_splat_profile.pgm` is a baked grayscale sprite profile used by `src/shaders/gas_splat.frag`.
- `reionization/star_glow_profile.pgm` is a baked grayscale sprite profile used by `src/shaders/star_glow.frag`.
